# Multi-App

Made by: <br>
Mambetkadyrov Salamatbek & Abdiev Atai & Mataraimov Nuradil <br>

⭒ "Compatibility Test" Game<br>
⭒ "Guess the Number" Game<br>
⭒ "Getter.io" Game<br>
